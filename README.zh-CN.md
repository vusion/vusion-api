## 安装

``` shell
npm install --save-dev ts-repo-boilerplate
```

## 选项

#### name

某个功能的名称。

- Type: `string`
- Default: `'hello'`

#### auto

是否自动做某件事。

- Type: `boolean`
- Default: `'true'`

## 开发

### dev

``` shell
npm run dev
```

### build

``` shell
npm run build
```

### test

